## 🚀 快速启动

### 1. 环境要求
- Node.js 18.x 或更高版本
- npm 或 yarn

### 2. 启动步骤
```bash
# 进入项目目录
cd webTryOn

# 一键启动（推荐）
./start-dev.sh

# 或手动启动
npm install
npm run dev
```

### 3. 访问应用
- **前端应用**: http://localhost:3000
- **后端API**: http://localhost:3001
- **健康检查**: http://localhost:3001/health
